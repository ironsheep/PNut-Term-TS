## Insights

### Why Such Dramatic Improvements?

1. **Pattern Recognition**: Once a pattern was established (e.g., WindowRouter), similar implementations were near-instant
2. **Code Generation**: Claude excels at generating boilerplate, tests, and documentation
3. **No Context Switching**: Claude maintains full project context without human memory limitations
4. **Parallel Processing**: Can work on multiple aspects simultaneously
5. **No Fatigue**: Consistent performance whether 1st or 34th task

### Estimation Patterns

Claude tends to:
- **Over-estimate familiar patterns**: Standard implementations estimated at hours, completed in minutes
- **Under-estimate novel integration**: Hardware/CLI integration took longer than expected
- **Accurately estimate research**: Exploration tasks matched estimates
